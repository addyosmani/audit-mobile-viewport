> Regex for checking HTML has a mobile meta-viewport


## Install

```sh
$ npm install --save audit-mobile-viewport
```


## Usage

```js
var audit-mobile-viewport = require('audit-mobile-viewport');

audit-mobile-viewport('Rainbow');
```
